Tentang Saya

Saya bekerja di bidang pelayanan kesehatan, dan di luar pekerjaan saya memiliki ketertarikan besar pada Bitcoin serta teknologi blockchain.

Selama lebih dari satu tahun terakhir, saya secara konsisten mempelajari Bitcoin, mulai dari sejarah, filosofi, mekanisme Proof of Work, mining (solo maupun pool), UTXO, Lightning Network, hingga keamanan aset digital melalui self-custody.

Fokus pembelajaran saya meliputi:

  • Bitcoin Self-Custody & Hardware Wallet Security
  • UTXO Management & Coin Control
  • Lightning Network
  • Mining & Mining Pool
  • On-chain Analysis
  • Privasi dan Keamanan Bitcoin
  • Bitcoin Improvement Proposal (BIP)

Saya percaya bahwa memahami cara kerja Bitcoin jauh lebih penting daripada sekadar membeli dan menyimpannya. Karena itu saya terus belajar, bereksperimen, dan membangun berbagai proyek kecil untuk memperdalam pemahaman saya mengenai Bitcoin dan teknologi open-source.

Jika perjalanan belajar ini bermanfaat atau menginspirasi, dukungan Anda akan sangat berarti untuk membantu saya terus belajar, membuat konten edukasi, dan membangun tools yang berguna bagi komunitas Bitcoin.
